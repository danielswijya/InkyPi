This plugin displays your custom bento grid dashboard with multiple widgets:

  • Weather (top left)
  • To-Do List (bottom left)
  • Image Slideshow (top middle) - Upload images via image_upload plugin
  • Sports Scores (middle)
  • Bible Quote (bottom middle)
  • Custom widgets (right column)

Note: The dashboard automatically pulls data from your configured plugins.

Preview your bento grid at: http://your-ip:8080/bento
